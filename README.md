# 图书进销货管理系统

## 项目简介
本项目为《数据库系统》课程设计，实现简易的图书进销货管理系统，支持多角色（系统管理员、财务人员、采购员、仓库管理员、销售员）协同管理图书的采购、库存、销售及财务统计等业务流程。系统基于 Python 开发，使用 GaussDB 作为数据库，使用 Flask 作为后端框架，前端页面采用 HTML 模板渲染。

## 主要功能
- 用户登录与权限管理
- 图书信息管理与库存管理
- 采购订单管理与供应商管理
- 销售订单管理与销售统计
- 财务审核与进销存统计
- 低库存预警与畅销书排行

## 目录结构
```
├── app.py                  # 主程序入口
├── db.py                   # 数据库操作模块
├── requirements.txt        # Python依赖包列表
├── templates/              # 前端HTML模板
│   ├── login.html
│   ├── admin_dashboard.html
│   ├── ...
├── pic/                    # 图片资源
│   ├── background/         # 角色背景图片
│   └── book/               # 图书封面图片
└── ...
```

## 安装与运行
1. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
2. 启动项目：
   ```bash
   python app.py
   ```
3. 访问系统：
   在浏览器中打开 `http://127.0.0.1:5000/`

## 注意事项
- 由于本项目使用 GaussDB 作为数据库，并采用弹性公网 IP 形式连接，因此在运行项目前需要确保已拥有华为云账号，并在控制台中创建相应的数据库实例及完成相应配置，具体步骤可参考华为云官方文档。
